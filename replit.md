# Sistema de Carteirinhas

## Overview
A card/badge generation system (Sistema de Carteirinhas) built with React, TypeScript, Vite, and Tailwind CSS. Uses Konva for canvas-based card template editing and rendering.

## Project Architecture
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS 3 + PostCSS
- **Canvas**: Konva + react-konva for card template editing
- **Port**: 5000 (dev server)

## Project Structure
- `src/App.tsx` - Main application component
- `src/main.tsx` - Entry point
- `src/index.css` - Global styles (Tailwind directives)
- `index.html` - HTML template
- `vite.config.ts` - Vite configuration
- `tailwind.config.js` - Tailwind configuration
- `postcss.config.js` - PostCSS configuration

## Running
- `npm run dev` - Start development server on port 5000
- `npm run build` - Build for production
